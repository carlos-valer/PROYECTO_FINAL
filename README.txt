BASE DE DATOS

Existen tres tipos de usuario:
- Alumno
- Profesor
- Director


+ Alumno :
el alumno tiene las opciones de:
 - modificar su clave
 - ver datos


+ Profesor :
el profesor tiene las opciones de:
 - modificar su clave
 - modificar notas de alumno
 - ver datos

+ Director :
el director tiene las opciones de:
 - modificar su clave
 - inscribir alumno
 - inscribir profesor
 - buscar alumnos
 - buscar profesores
 - ver datos
